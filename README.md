# Manus Skills Collection

Coleção curada de 20 skills essenciais para o Manus, selecionadas para **Medicina**, **Marketing Digital** e **Educação**.

## Skills Disponíveis

### Medicina e Saúde (6 skills)

| Skill | Descrição |
| --- | --- |
| `clinical-reports` | Gera relatórios clínicos (SOAP, CARE, ICH-E3, resumos de alta) |
| `clinical-decision-support` | Suporte à decisão clínica para pesquisa e prática |
| `treatment-plans` | Auxilia na criação de planos de tratamento |
| `pubmed-database` | Pesquisa na base de dados PubMed |
| `drugbank-database` | Informações sobre medicamentos e interações |
| `clinicaltrials-database` | Consulta ensaios clínicos no ClinicalTrials.gov |

### Marketing Digital (4 skills)

| Skill | Descrição |
| --- | --- |
| `twitter-algorithm-optimizer` | Otimiza tweets para maior alcance e engajamento |
| `competitive-ads-extractor` | Extrai e analisa anúncios de concorrentes |
| `lead-research-assistant` | Identifica e qualifica leads de alta qualidade |
| `content-research-writer` | Pesquisa e escreve conteúdo otimizado |

### Educação e Cursos (5 skills)

| Skill | Descrição |
| --- | --- |
| `flashcards` | Cria cartões de memorização |
| `mindmap` | Transforma anotações em mapas mentais |
| `quiz` | Gera questionários práticos com explicações |
| `scientific-writing` | Escrita científica e acadêmica |
| `literature-review` | Revisão sistemática de literatura |

### Documentos e Produtividade (5 skills)

| Skill | Descrição |
| --- | --- |
| `pdf` | Manipulação de arquivos PDF |
| `docx` | Criação e edição de documentos Word |
| `pptx` | Criação de apresentações PowerPoint |
| `xlsx` | Manipulação de planilhas Excel |
| `perplexity-search` | Pesquisa avançada com Perplexity AI |

## Como Usar

Para usar uma skill no Manus, basta solicitar:

```
Use a skill [nome-da-skill]
```

**Exemplos:**
- `Use a skill clinical-reports para criar um relatório SOAP`
- `Use a skill twitter-algorithm-optimizer para otimizar meu tweet`
- `Use a skill flashcards para criar cartões de estudo`
